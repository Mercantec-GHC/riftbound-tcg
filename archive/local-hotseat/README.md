# Local Hot-Seat Prototype Archive

This folder preserves the pre-API local hot-seat prototype code after the active frontend moved cards, decks, accounts, and match state to the backend API.

The files here are not imported by the active Vite app and should not be treated as production gameplay authority. They are retained so the local prototype UI, local storage helpers, and `gameRules.ts` behavior remain available for reference or future extraction.

Rules work is intentionally out of scope for this archive. Do not update these files unless the archived prototype itself is being restored or documented.
